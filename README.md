# JavaScript Basics

## Overview

This repository contains **JavaScript fundamentals** and basic programming concepts. It includes examples and exercises for learning JavaScript programming basics.

## Contents

- **Variables & Data Types:** Strings, numbers, booleans, objects, arrays
- **Functions:** Function declarations, arrow functions, callbacks
- **Control Flow:** If-else, loops, switch statements
- **DOM Manipulation:** Selecting, creating, modifying HTML elements
- **Event Handling:** Click, input, form events
- **ES6 Features:** Arrow functions, destructuring, template literals
- **Async Programming:** Promises, async/await basics

## Key Topics

### Fundamentals
- Variable declaration (var, let, const)
- Data types and type coercion
- Operators and expressions
- String methods and manipulation
- Array methods (map, filter, reduce)

### Functions
- Function declaration and expression
- Arrow functions
- Closure and scope
- Higher-order functions

### DOM
- Selecting elements (getElementById, querySelector)
- Creating and removing elements
- Modifying styles and attributes
- Event listeners

### Modern JavaScript
- Template literals
- Destructuring assignment
- Spread operator
- Classes and prototypes

## Getting Started

### Prerequisites
- Web browser with JavaScript support
- Code editor
- Basic HTML knowledge

### Running Examples

1. Open HTML files in a web browser
2. Check browser console for output (F12 or Ctrl+Shift+I)
3. Modify and experiment with code

## Learning Path

1. Start with basics (variables, operators)
2. Learn control flow (if, loops)
3. Master functions
4. Explore DOM manipulation
5. Practice event handling
6. Learn ES6+ features
7. Understand async operations
